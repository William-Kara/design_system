# Welcome to the Design System

## Description

- **Foundation**: Our colors, grid, typo and basic visual styles
- **Elements**: Banner, Box, Button, Link, Nav, Separator, Tag, Textfield
- **Modules**: Card, CardSlider, Footer, Header, Modal, TagList
- **Prototype**: BlockContentTitle

## Usage

### How to use the design system ?

### Outside spacing

Design system components (Elements, Modules & Prototypes) have no outside spacing (**no margin**). Outside spacing has to be handle outside the Design Sytem. Margin should be set using bulma classes [as per as Spacings foundations](/docs/foundations-spacings--docs)

### Layout positioning

Bulma grid and spacing should be used to position element on the layout as much as possible. If needed specific css can be added to position element on the layout.

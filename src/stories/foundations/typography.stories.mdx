# Typography

Interligne should be 1.5 .

## Body text

Body text in paragraph should be in :

**Font:** Poppins, Regular
**Size:** 1.12rem, 1.25rem

<Typeset
  fontSizes={[Poppins.size.s, Poppins.size.m]}
  fontWeight={Poppins.weight.regular}
  sampleText={SampleText}
  fontFamily={Poppins.type.primary}
/>

Text in input should be in :

**Font:** Poppins, Medium
**Size:** 1rem

<Typeset
  fontSizes={[Poppins.size.xs]}
  fontWeight={Poppins.weight.medium}
  sampleText={SampleText}
  fontFamily={Poppins.type.primary}
/>

A extra small font is also available :

**Font:** Poppins, Medium
**Size:** 0.6rem

<Typeset
  fontSizes={[Poppins.size.xxs]}
  fontWeight={Poppins.weight.regular}
  sampleText={SampleText}
  fontFamily={Poppins.type.primary}
/>

## Titles

Bigger font sizes are used for titles.

## Card Title

Card titles used :

**Font:** Poppins, light
**Size:** 1.75rem

<Typeset
  fontSizes={[Poppins.size.m2]}
  fontWeight={Poppins.weight.light}
  sampleText={'Card Title'}
  fontFamily={Poppins.type.primary}
/>

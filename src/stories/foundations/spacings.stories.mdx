# Spacings

This Design System used the [Bulma Spacing Helpers](https://bulma.io/documentation/helpers/spacing-helpers/).

Spacing unit is equal to 0.25rem, which translates to 4px by default in common browsers.

Padding & Margin classes examples:

```css
.m-0 {
  margin: 0 !important;
}

.p-1 {
  padding: 0.25rem !important;
}

.py-2 {
  padding-top: 0.5rem !important;
  padding-bottom: 0.5rem !important;
}

.px-4 {
  padding-left: 1rem !important;
  padding-right: 1rem !important;
}

.pt-6 {
  padding-top: 3rem !important;
}

.mt {
  margin-top: auto !important;
}
```

# Greys

There are 7 shades of grey, ranging from black (darkest) to grey-alto (lightest).

Some of the use cases are listed below:

- **black**: background, button labels
- **black-lighten**: banners
- **grey-mineshaft**: cards
- **black-hover**: button hover
- **grey-emperor**: tags, disabled buttons
- **grey-boulder**: button borders
- **grey-manatee**: button disabled labels
- **grey-alto**: button disabled labels

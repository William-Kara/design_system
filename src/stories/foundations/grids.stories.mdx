# Grids

The grids system used for this website is based on the [Bulma Section](https://bulma.io/documentation/helpers/spacing-helpers/) and [Bulma Container](https://bulma.io/documentation/helpers/spacing-helpers/) for gutters, breakpoints and max-width and on the [Bulma Columns classes](https://bulma.io/documentation/helpers/spacing-helpers/) for columns layouts.

## Breaking points

The breaking points are:

- mobile: up to 768px
- tablet: from 769px
- desktop: from 1024px
- widescreen: from 1216px
- fullhd: from 1408px

## Section

The section components are simple layout elements with responsive padding that must be used as direct children of the body.

The default padding applied to the section components is like so:

```css
padding: 3rem 3rem;
```

## Container

The container is a simple utility element that allows you to center content on the viewports.

This is how the container will behave:

- on Desktop it will have a maximum width of 960px.
- on Widescreen it will have a maximum width of 1152px.
- on Fullhd it will have a maximum width of 1344px.

## Unequal Columns layout

If the layout requires unequal columns layout, you may need to use the following classes on a single column:

- is-three-quarters
- is-two-thirds
- is-half
- is-one-third
- is-one-quarter
- is-full
- is-four-fifths
- is-three-fifths
- is-two-fifths
- is-one-fifth

The other columns will fill up the remaining space automatically.

<div class="columns">
  <div class="column is-three-quarters">
    <p class="example-col">is-three-quarters</p>
  </div>
  <div class="column is-one-quarter">
    <p class="example-col">is-one-quarter</p>
  </div>
</div>

```html
<div class="columns">
  <div class="column is-three-quarters">is-three-quarters</div>
  <div class="column is-one-quarter">is-one-quarter</div>
</div>
```

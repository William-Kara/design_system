# Usage

To use QfcBox, style needs to be imported separately.

```sass
@import '@website/design-system-vuejs/dist/styles/bulma/elements/box.scss';
@import '@website/design-system-vuejs/dist/styles/components/QfcBox.scss';
```

```html
<template>
  <QfcBox size="medium">
    <p>Lorem ipsum dolor sit amet, consectetur adipisicing elit.</p>
  </QfcBox>
</template>
```

## Usage

To use QfcSubtitle, style needs to be imported separately.

```sass
@import '@website/design-system-vuejs/dist/styles/components/QfcSubtitle.scss';

```

```html
<template>
  <QfcSubtitle :text="Subtitle" visualLevel="1" />
</template>
```

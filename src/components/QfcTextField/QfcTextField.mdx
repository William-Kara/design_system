# Definition

A customizable text field component This component is a text field that can be used for various types of input, such as text, password, email and number.

## Props

The following props are available for the QfcTextField component:

<ArgTypes of={QfcTextField} />
When the isRequired prop is set to true, an asterisk appears to the right of the label.

An eye icon to toggle the visibility of the input when the type is set to “password” is available.

The input field become red when the errorMessage prop is set. But also when the input get the :invalid pseudo-class that can happen when the input is required and empty or when the input doesn't match the pattern asked.

# Usage

To use QfcTextField, style needs to be imported separately.

```sass
@import '@website/design-system-vuejs/dist/styles/components/QfcTextField.scss';

```

```html
<template>
  <QfcTextField label="Name" :isRequired="true" />
</template>
```

## Background color

The background color of the input field can be customized by using hexadecimal color codes.

```html
<template>
  <QfcTextField label="Name" :isRequired="true" backgroundColor="#f5f5f5" />
</template>
```

The background color act as background for textfield and the label moving up when the input is focused.

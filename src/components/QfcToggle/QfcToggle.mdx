# Definition & Usage

QfcToggle is a toggle button that should be use for boolean value.

## Props

The following props are available for the QfcToggle component:

<ArgTypes of={QfcToggle} />

## Usage

To use QfcToggle, style needs to be imported separately.

```sass
@import '@website/design-system-vuejs/dist/styles/components/QfcToggle.scss';

```

```html
<template>
  <QfcToggle v-model="myModel" :disabled="isDisabled" label="Toggle Label" />
</template>
```

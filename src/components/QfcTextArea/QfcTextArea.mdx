## Background color

The background color of the input field can be customized by using hexadecimal color codes.

```html
<template>
  <QfcTextArea label="Name" :isRequired="true" backgroundColor="#f5f5f5" />
</template>
```

The background color act as background for textfield and the label moving up when the input is focused.

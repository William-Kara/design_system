# Definition

The QfcButton is clickable element which can be used in forms, or anywhere that needs simple validation functionality.

This button may display text with ou without icons and has different appearances. Use a "Primary" button to call attention to the main action. Don't use multiple primary buttons on one page and use a "Secondary" button for secondary actions, such as â€œCancel".

# Usage

To use QfcButton, style needs to be imported separately.

```sass
@import '@website/design-system-vuejs/dist/styles/bulma/elements/button.scss';
@import '@website/design-system-vuejs/dist/styles/components/QfcButton.scss';
```

```html
<template>
  <QfcButton label="Button1" isDisabled="true" secondary="true" icon="star" />
</template>
```

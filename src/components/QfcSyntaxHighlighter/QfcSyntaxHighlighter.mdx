# Definition

QfcSyntaxHighlighter allow you to display a block code / synthax highlight block

# Import

```sass
@import '@website/design-system-vuejs/dist/styles/components/QfcSyntaxHighlighter.scss';
```

# Usage

```html
<template>
  <pre>
        <code class="language-javascript">
          // code here for example
          function hello(){
             console.log(Hello World)
          }
        </code>
    </pre>
</template>
```
